What is AnyText?

AnyText is an AI-powered browser extension that enhances any text input field with intelligent writing tools. Right-click on selected text to access powerful AI features.

How to Use

  1. Select any text on a webpage or the icon inside a input field
  2. Right-click to open the context menu
  3. Choose "AnyText" and select your desired action
  4. The AI-enhanced text will replace your selection

Requirements

AnyText uses Chrome's Built-in AI APIs.
